# app-restguard test/app

> Devebot access-token authentication layerware

## Usage

### Run the server for testing

```shell
export LOGOLITE_DEBUGLOG_ENABLED=true
export DEBUG=devebot*,app*
npm run clean
npm run build && node test/app
```
